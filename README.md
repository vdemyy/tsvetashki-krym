🌸 ЦВЕТАШКИ КРЫМ
Сезонные явления полуострова — веб-приложение и Telegram бот

================================================================================

📌 О проекте

   Веб-приложение и Telegram-бот для отслеживания сезонных явлений в Крыму.

   Что можно отслеживать:

   ┌─────────────────────────┬────────────────────────────────────────────────┐
   │ 🌺 Цветение             │ лаванда, сакура, маки, пионы, глициния         │
   ├─────────────────────────┼────────────────────────────────────────────────┤
   │ 🌅 Визуальные эффекты   │ закаты с зелёным лучом, туманы, гало           │
   ├─────────────────────────┼────────────────────────────────────────────────┤
   │ 🍇 Урожай               │ черешня, виноград                              │
   ├─────────────────────────┼────────────────────────────────────────────────┤
   │ 🐬 Животные             │ дельфины у берега                              │
   ├─────────────────────────┼────────────────────────────────────────────────┤
   │ 🎪 Мероприятия          │ фестивали вина, ярмарки мёда                    │
   └─────────────────────────┴────────────────────────────────────────────────┘

⚙️ ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ (.env)

   Создайте файл .env в корне проекта:

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ # Обязательные                                                              │
   │ SESSION_SECRET=ваша-случайная-строка-из-32-символов                         │
   │ ADMIN_PASSWORD=ваш-пароль-для-админки                                       │
   │ DATABASE_URL=sqlite:///./data/tsvetashki.db                                 │
   │                                                                             │
   │ # Опциональные                                                              │
   │ OPENWEATHER_API_KEY=                                                        │
   │ TELEGRAM_BOT_TOKEN=                                                         │
   │ BASE_URL=http://localhost:8000                                              │
   └─────────────────────────────────────────────────────────────────────────────┘

   🔑 Как сгенерировать SESSION_SECRET:

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ python -c "import secrets; print(secrets.token_urlsafe(32))"               │
   └─────────────────────────────────────────────────────────────────────────────┘
   [Нажмите на команду, чтобы скопировать]

🚀 ЗАПУСК

   ┌─ Способ 1: Локальный запуск ─────────────────────────────────────────────────┐
   │                                                                              │
   │ git clone https://github.com/videmyy/tsvetashki-krym.git                     │
   │ cd tsvetashki-krym                                                           │
   │ python -m venv venv                                                          │
   │ venv\Scripts\activate      # Windows                                         │
   │ source venv/bin/activate   # Linux/Mac                                       │
   │ pip install -r requirements.txt                                              │
   │ copy .env.example .env      # Windows                                        │
   │ cp .env.example .env        # Linux/Mac                                      │
   │ uvicorn main:app --reload --port 8000                                        │
   │                                                                              │
   └──────────────────────────────────────────────────────────────────────────────┘

   ┌─ Способ 2: Запуск через Docker ───────────────────────────────────────────────┐
   │                                                                              │
   │ docker compose up backend                                                    │
   │ docker compose --profile bot up                                              │
   │ docker compose down                                                          │
   │                                                                              │
   └──────────────────────────────────────────────────────────────────────────────┘

🐛 РЕШЕНИЕ ПРОБЛЕМ

   1️⃣ База данных не создаётся

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ python -c "from main import app, lifespan; import asyncio; asyncio.run(lifespan(app))" │
   └─────────────────────────────────────────────────────────────────────────────┘

   2️⃣ Порт 8000 уже занят (Windows)

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ netstat -ano | findstr :8000                                                │
   │ taskkill /PID <PID> /F                                                      │
   └─────────────────────────────────────────────────────────────────────────────┘

   3️⃣ Docker контейнеры не запускаются

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ docker compose down -v                                                      │
   │ docker compose build --no-cache                                             │
   │ docker compose up                                                           │
   └─────────────────────────────────────────────────────────────────────────────┘

   4️⃣ Telegram бот не отвечает

   ┌─────────────────────────────────────────────────────────────────────────────┐
   │ # Проверьте токен в .env файле                                              │
   │ TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather                                   │
   │                                                                             │
   │ # Проверьте логи бота                                                       │
   │ docker compose logs bot                                                     │
   └─────────────────────────────────────────────────────────────────────────────┘
