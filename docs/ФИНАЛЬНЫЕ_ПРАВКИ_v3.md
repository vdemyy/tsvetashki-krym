## 1. Хедер на мобильных - только иконки

**Проблема**: Кнопки с текстом занимали слишком много места, не помещались.

**Решение**:
- Убран весь текст из кнопок навигации на мобильных
- `font-size: 0` скрывает текст
- Увеличены иконки до 1.1rem для лучшей видимости
- Padding: 0.5rem (квадратные кнопки)
- Кнопка темы: 2.2rem × 2.2rem

**Результат**: Компактный хедер, все кнопки помещаются на одной строке с логотипом.

**Файл**: `static/css/style.css` (@media max-width: 640px)

```css
.nav__link {
  padding: 0.5rem;
  font-size: 0; /* Скрывает текст */
  white-space: nowrap;
}
.nav__ico {
  width: 1.1rem;
  height: 1.1rem;
  margin: 0;
}
```

## 2. Кнопки "7 дней", "Шире", "Карта" - серый текст

**Проблема**: Белый текст на белом фоне был не виден.

**Решение**:
- **Неактивные кнопки**: серый текст `#5e6560`
- **При наведении**: зеленый текст `var(--accent)`
- **Активная кнопка**: белый текст на зеленом фоне
- **Ghost кнопки**: серый текст, прозрачный фон

**Светлая тема**:
- Неактивные: серый текст (#5e6560) на белом фоне
- Hover: зеленый текст на светло-сером фоне
- Активные: белый текст на зеленом фоне

**Темная тема**:
- Неактивные: серый текст (#8b949e) на синем полупрозрачном фоне
- Hover: синий текст (#58a6ff)
- Активные: белый текст на синем фоне

**Файл**: `static/css/style.css` (секция .pill)

```css
.pill {
  color: #5e6560; /* Серый текст */
}
.pill:hover {
  color: var(--accent); /* Зеленый при наведении */
}
.pill--on {
  color: #fff; /* Белый на активной */
}
```

## 3. Уменьшены отступы между фильтрами и карточками

**Проблема**: Слишком большое расстояние между фильтрами и первой карточкой.

**Решение**:
- `.panel { margin-bottom: 1rem }` (было 1.35rem)
- На мобильных:
  - `.filters-toggle { margin-bottom: 0.5rem }` (было 0.75rem)
  - `.filters.active { margin-bottom: 0.5rem }` (было 0.75rem)

**Результат**: Компактная страница, меньше прокрутки.

**Файл**: `static/css/style.css`
