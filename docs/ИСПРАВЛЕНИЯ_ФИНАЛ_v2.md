# Финальные исправления v2

## Дата: 21 мая 2026

### Исправленные проблемы:

#### 1. ✅ Хедер на мобильных устройствах
**Проблема**: Кнопки навигации были не на одном уровне с логотипом, хедер выглядел плохо.

**Решение**:
- Использован CSS Grid для выравнивания
- Логотип и навигация теперь на одном уровне (grid-row: 1)
- Tagline на отдельной строке внизу (grid-row: 2)
- Логотип уменьшен до 2.5rem для лучшего баланса
- Текст логотипа: 1rem

**Файлы**: `static/css/style.css` (медиа-запрос @media (max-width: 640px))

---

#### 2. ✅ Отступ между фильтрами и карточками
**Проблема**: Слишком большой отступ между кнопкой "Фильтры" и первой карточкой.

**Решение**:
- Уменьшен `margin-bottom` с 1rem до 0.75rem для `.filters-toggle`
- Уменьшен `margin-bottom` с 1rem до 0.75rem для `.filters.active`

**Файлы**: `static/css/style.css`

---

#### 3. ✅ Видимость текста на кнопках навигации
**Проблема**: Текст на кнопках "Лента", "Карта", "Вход" был плохо виден.

**Решение**:
- Изменен цвет текста с `rgba(255,255,255,0.95)` на `#ffffff` (полностью белый)
- Добавлена тень текста: `text-shadow: 0 1px 2px rgba(0,0,0,0.2)`
- Добавлена граница: `border: 1px solid rgba(255,255,255,0.2)`
- Иконки теперь явно белые: `color: #ffffff`

**Файлы**: `static/css/style.css` (секция .nav__link)

---

#### 4. ✅ "Облачность" больше не кнопка
**Проблема**: "Облачность" на карте была кнопкой, а должна быть переключателем слоя.

**Решение**:
- Заменена кнопка на `<label>` с чекбоксом
- Добавлены стили `.map-ctrl-toggle` для переключателя
- Чекбокс скрыт визуально, но работает функционально
- При включении фон меняется на `var(--accent-soft)` с зеленой границей
- JavaScript обновлен для работы с `change` событием чекбокса

**Файлы**: 
- `templates/map.html` (HTML структура)
- `static/js/map.js` (логика переключения)

---

#### 5. ✅ Анимации на десктопе
**Проблема**: На десктопе анимации не работали правильно.

**Решение**:
- Добавлены console.log для отладки в `theme.js`
- Логика осталась прежней, но теперь можно отследить, что происходит
- Версия кеша увеличена до `v=4` для принудительной перезагрузки

**Файлы**: 
- `static/js/theme.js`
- `templates/base.html` (версия v=4)

### Технические детали

```css
@media (max-width: 640px) {
  .site-header__inner {
    display: grid;
    grid-template-columns: 1fr auto;
    grid-template-rows: auto auto;
  }
  .logo {
    grid-column: 1 / 2;
    grid-row: 1 / 2;
  }

    grid-column: 2 / 3;
    grid-row: 1 / 2;
  }
  .tagline {
    grid-column: 1 / -1;
    grid-row: 2 / 3;
  }
}
```
