## ✅ Исправлено:

### 1. Убран текст "Цветение Крыма" из хедера

**Проблема**: Текст занимал место в хедере.

**Решение**:
- Удален `<span class="logo__text">Цветение Крыма</span>`
- Остался только логотип (изображение)
- Хедер стал компактнее

**Файл**: `templates/base.html`

**До:**
```html
<a class="logo" href="/">
  <img src="/img/logo.png" alt="Цветение Крыма" class="logo__img">
  <span class="logo__text">Цветение Крыма</span>
</a>
```

**После:**
```html
<a class="logo" href="/">
  <img src="/img/logo.png" alt="Цветение Крыма" class="logo__img">
</a>
```

### 2. Кнопки "7 дней", "Шире", "Карта" - темно-серый текст

**Проблема**: Серый текст (#5e6560) был недостаточно контрастным на белом фоне.

**Решение**:
- Изменен цвет с `#5e6560` на `#3a3d3a` (темно-серый)
- **Светлая тема**: темно-серый текст (#3a3d3a) - **отлично виден!**
- **Темная тема**: светло-серый текст (#c9d1d9) - **хорошо виден!**

**Файл**: `static/css/style.css`

**Цвета:**
```css
/* Светлая тема */
.pill {
  color: #3a3d3a; /* Темно-серый - хорошо виден на белом */
}

/* Темная тема */
[data-theme="dark"] .pill {
  color: #c9d1d9; /* Светло-серый - хорошо виден на темном */
}
```
