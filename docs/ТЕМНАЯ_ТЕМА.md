# 🌙 Темная тема

## ✨ Что добавлено

### 1. Темная и светлая темы
- **Светлая тема** (по умолчанию) - классический дизайн с теплыми тонами
- **Темная тема** - современный темный дизайн для комфортного просмотра вечером

### 2. Переключатель темы
- Кнопка в хедере с иконками солнца/луны
- Плавная анимация переключения
- Сохранение выбранной темы в localStorage

### 3. Автоматическое переключение по времени
- **Темная тема** включается автоматически после заката
- **Светлая тема** включается автоматически на рассвете
- Использует данные о времени заката/рассвета из OpenWeather API
- Координаты: Симферополь (центр Крыма)

### 4. Анимации по теме
- **Светлая тема**: лепестки сакуры 🌸
- **Темная тема**: светлячки ✨
- Анимации автоматически переключаются вместе с темой

## 🔧 Как это работает

### 1. Определение темы

Тема определяется в следующем порядке:

1. **Автоматически по времени** (если включено):
   - Запрос к API `/api/sun-times` для получения времени заката/рассвета
   - Сравнение текущего времени с временем заката/рассвета
   - Автоматическое переключение темы

2. **Вручную** (если пользователь нажал кнопку):
   - Переключение темы по клику
   - Отключение автоматического режима
   - Сохранение выбора в localStorage

3. **Из localStorage** (при загрузке страницы):
   - Восстановление последней выбранной темы

### 2. API endpoint для времени заката/рассвета

**Endpoint**: `GET /api/sun-times?lat={latitude}&lon={longitude}`

**Пример запроса**:
```bash
curl "http://localhost:8000/api/sun-times?lat=44.9572&lon=34.1108"
```

**Ответ**:
```json
{
  "sunrise": 1716267600,
  "sunset": 1716318000,
  "timezone": 10800
}
```

**Параметры**:
- `lat` - широта (для Симферополя: 44.9572)
- `lon` - долгота (для Симферополя: 34.1108)

**Возвращает**:
- `sunrise` - время рассвета (Unix timestamp)
- `sunset` - время заката (Unix timestamp)
- `timezone` - часовой пояс (секунды от UTC)

### 3. JavaScript логика

**Файл**: `static/js/theme.js`

**Основные функции**:
- `initTheme()` - инициализация темы при загрузке
- `toggleTheme()` - переключение темы вручную
- `applyTheme(theme)` - применение темы
- `fetchSunTimes()` - получение времени заката/рассвета
- `getThemeByTime()` - определение темы по времени
- `applyAutoTheme()` - автоматическое применение темы
- `updateAnimations(theme)` - переключение анимаций

**Хранение данных**:
- `localStorage.theme` - выбранная тема ('light' или 'dark')
- `localStorage.autoTheme` - включен ли автоматический режим ('true' или 'false')

## 🎭 Анимации

### Лепестки сакуры (светлая тема)

**Файл**: `static/js/petals.js`

**Функции**:
- `startPetals()` - запуск анимации
- `stopPetals()` - остановка анимации
- `createPetal()` - создание одного лепестка

**Параметры**:
- Цвета: розовые оттенки (#ffc0d3, #ffb3c6, #ffd4e5, #ffe0ec, #f8b4d9)
- Размер: 12-28px
- Длительность: 8-15 секунд
- Горизонтальное покачивание: 40-120px
- Вращение: 2-5 полных оборотов

### Светлячки (темная тема)

**Файл**: `static/js/fireflies.js`

**Функции**:
- `startFireflies()` - запуск анимации
- `stopFireflies()` - остановка анимации
- `createFirefly()` - создание одного светлячка

**Параметры**:
- Цвета: синие и фиолетовые оттенки (#58a6ff, #79c0ff, #d2a8ff, #f778ba, #b4dcff)
- Размер: 2-6px
- Длительность: 15-35 секунд
- Горизонтальный дрейф: -100 до +200px
- Свечение: box-shadow с цветом светлячка

   - Скрыт tagline
   - Уменьшен хедер

## 🚀 Использование

### Ручное переключение темы

Нажмите кнопку с иконкой солнца/луны в хедере.

### Автоматический режим

По умолчанию включен. Тема будет переключаться автоматически:
- **Темная** - после заката
- **Светлая** - после рассвета

### Отключение автоматического режима

Просто нажмите кнопку переключения темы вручную. Автоматический режим отключится, и ваш выбор сохранится.

### Включение автоматического режима обратно

Очистите localStorage:
```javascript
localStorage.removeItem('autoTheme');
location.reload();
```

Или в консоли браузера:
```javascript
localStorage.setItem('autoTheme', 'true');
location.reload();
```

Откройте консоль браузера (F12):
```javascript
document.documentElement.getAttribute('data-theme')
// Вернет: 'light' или 'dark'
```

### 2. Проверить время заката/рассвета

```bash
curl "http://localhost:8000/api/sun-times?lat=44.9572&lon=34.1108"
```

### 3. Проверить localStorage

```javascript
localStorage.getItem('theme')        // 'light' или 'dark'
localStorage.getItem('autoTheme')    // 'true' или 'false'
```

### 4. Принудительно установить тему

```javascript
// Светлая тема
document.documentElement.setAttribute('data-theme', 'light');

// Темная тема
document.documentElement.setAttribute('data-theme', 'dark');
```

## 🎨 Кастомизация

### Изменить цвета темы

Отредактируйте переменные в `static/css/style.css`:

```css
/* Светлая тема */
:root {
  --bg: #f7f4f0;
  --accent: #2d5a3d;
  /* ... */
}

/* Темная тема */
[data-theme="dark"] {
  --bg: #0f1410;
  --accent: #4a9d6a;
  /* ... */
}
```

### Изменить координаты для определения заката/рассвета

```javascript
// Ваши координаты

```

## 📊 Технические детали

### Файлы

```
static/
├── css/
│   └── style.css           # Стили для обеих тем
├── js/
│   ├── theme.js            # Логика переключения темы
│   ├── petals.js           # Анимация лепестков

templates/

routers/

```

### Зависимости

- **OpenWeather API** - для получения времени заката/рассвета
- **localStorage** - для сохранения выбора темы
- **CSS Variables** - для динамического изменения цветов
- **Lucide Icons** - иконки солнца и луны

### Производительность

- Анимации используют CSS animations (GPU-ускорение)
- Проверка времени каждую минуту (не каждую секунду)
- Lazy loading для анимаций
- Минимальное использование JavaScript
