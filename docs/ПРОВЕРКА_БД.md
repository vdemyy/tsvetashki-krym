## Команды для проверки

### 1. Проверить запущенные контейнеры
```bash
docker compose ps
```

Должны быть запущены:
- `tsvetashki_db` (PostgreSQL)
- `tsvetashki_backend` (FastAPI приложение)

### 2. Подключиться к PostgreSQL
```bash
docker compose exec db psql -U tsvetashki -d tsvetashki
```

### 3. Посмотреть таблицы
```bash
docker compose exec db psql -U tsvetashki -d tsvetashki -c "\dt"
```

### 4. Посмотреть количество записей
```bash
docker compose exec db psql -U tsvetashki -d tsvetashki -c "SELECT 'phenomena' as table_name, COUNT(*) FROM phenomena UNION ALL SELECT 'places', COUNT(*) FROM places UNION ALL SELECT 'events', COUNT(*) FROM events;"
```

### 5. Проверить, какую БД использует приложение
```bash
docker compose exec backend python -c "import os; print('DATABASE_URL:', os.getenv('DATABASE_URL'))"
```

Должно вывести:
```
DATABASE_URL: postgresql://tsvetashki:tsvetashki_pass_2026@db:5432/tsvetashki
```

### 6. Проверить данные через API
```bash
curl http://localhost:8000/api/events
```

## pgAdmin (опционально)

Если хотите использовать pgAdmin для управления БД, добавьте в `docker-compose.yml`:

```yaml
  pgadmin:
    image: dpage/pgadmin4:latest
    container_name: tsvetashki_pgadmin
    restart: unless-stopped
    environment:
      PGADMIN_DEFAULT_EMAIL: admin@tsvetashki.local
      PGADMIN_DEFAULT_PASSWORD: admin
    ports:
      - "5050:80"
    depends_on:
      - db
```

Затем:
1. Запустите: `docker compose up -d pgadmin`
2. Откройте: http://localhost:5050
3. Войдите: admin@tsvetashki.local / admin
4. Добавьте сервер:
   - Host: `db`
   - Port: `5432`
   - Database: `tsvetashki`
   - Username: `tsvetashki`
   - Password: `tsvetashki_pass_2026`

## SQLite больше не используется

Файл `data/tsvetashki.db` (SQLite) остался как резервная копия, но приложение его **не использует**.

Все новые данные сохраняются в **PostgreSQL**.

## Резервное копирование

### Создать бэкап
```bash
docker compose exec db pg_dump -U tsvetashki tsvetashki > backup_$(date +%Y%m%d_%H%M%S).sql
```

### Восстановить из бэкапа
```bash
docker compose exec -T db psql -U tsvetashki tsvetashki < backup_20260518_210000.sql
```
