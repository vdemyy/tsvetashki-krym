# 🎨 Обновления дизайна

## ✅ Что сделано

### 1. 🖼️ Увеличен логотип
- **Размер**: с 3.5rem до **4.5rem** (на ~29% больше)
- **Путь**: изменен с `/static/img/logo.png` на `/img/logo.png`
- **Текст логотипа**: увеличен с 1.35rem до **1.5rem**
- Логотип теперь более заметный и привлекает внимание

### 2. 🌙 Добавлена темная тема
- **Светлая тема** (по умолчанию) - классический дизайн
- **Темная тема** - современный дизайн с синей палитрой
- **Переключатель** в хедере с иконками солнца/луны
- **Автоматическое переключение** по времени заката/рассвета

#### Цветовая палитра темной темы:
- **Фон**: #0d1117 (темно-серый, как у GitHub)
- **Карточки**: #1c2128 (темные, не белые!)
- **Акцент**: #58a6ff (синий)
- **Дополнительный акцент**: #d2a8ff (фиолетовый)
- **Текст**: #e6edf3 (светлый, хорошая читаемость)

### 3. ✨ Анимации по теме
- **Светлая тема**: лепестки сакуры 🌸 (розовые оттенки)
- **Темная тема**: светлячки ✨ (синие и фиолетовые оттенки)
- Анимации автоматически переключаются вместе с темой

### 4. 🗺️ Исправлены кнопки на карте
- Кнопки управления больше **не накладываются** друг на друга
- Leaflet controls смещены вниз (top: 70px на десктопе, 60px на мобильных)
- Кнопки адаптированы под темную тему
- Легенда карты также поддерживает обе темы

### 5. 📱 Улучшена мобильная версия

#### Общие улучшения:
- Адаптивные отступы и размеры
- Компактный хедер
- Оптимизированные карточки
- Улучшенная читаемость

#### Для экранов <640px:
- Логотип: 3.8rem
- Компактная навигация
- Уменьшенные кнопки
- Адаптивная сетка дат

#### Для экранов <380px:
- Скрыты текстовые метки в навигации (только иконки)
- Логотип: 3.2rem
- Вертикальное расположение элементов

#### Ландшафтная ориентация:
- Скрыт tagline
- Минимальный хедер
- Карта занимает максимум места

## 🔍 Как проверить

### 1. Проверить логотип
- Откройте http://localhost:8000
- Логотип должен быть большим и заметным
- Путь в HTML: `/img/logo.png`

### 2. Проверить темную тему
- Нажмите кнопку с луной в хедере
- Фон должен стать темным (#0d1117)
- Карточки должны быть темными (#1c2128), не белыми
- Акцентный цвет - синий (#58a6ff)

### 3. Проверить анимации
- **Светлая тема**: должны падать розовые лепестки сакуры
- **Темная тема**: должны летать синие/фиолетовые светлячки

### 4. Проверить карту
- Откройте http://localhost:8000/map
- Кнопка "Облачность" не должна накладываться на Leaflet controls
- Легенда должна быть видна и адаптирована под тему

### 5. Проверить мобильную версию
- Откройте DevTools (F12)
- Переключите на мобильное устройство (Ctrl+Shift+M)
- Проверьте разные размеры экрана:
  - 640px - компактный дизайн
  - 380px - только иконки в навигации
  - Ландшафт - минимальный хедер

### 6. Проверить автоматическое переключение
```bash
# Получить время заката/рассвета
curl "http://localhost:8000/api/sun-times?lat=44.9572&lon=34.1108"
```
